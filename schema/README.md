# Manifest JSON Schema

This directory contains a copy of the OAA 0.1 manifest JSON Schema used by the validator.

The authoritative schema is maintained in the [OAA specification repository](https://github.com/Original-Art-Archive/oaa-spec/blob/main/schema/oaa-manifest.schema.json).

[oaa-manifest.schema.json](oaa-manifest.schema.json) is used for manifest-local JSON structure: field presence, JSON types, and constraints expressible within a single manifest document.

The validator then applies additional archive-wide and cross-reference validation rules.
